Museum Geopark Batur

Geopark Batur

Menyajikan tiga pilar pendukung geopark berdasarkan sejarah geologi, lingkungan, ekosistem, dan interaksi sosial dengan fokus adaptasi dan keragaman.

Ruang Pengenalan

Memuat informasi tentang pengertian Taman Bumi (Geopark), Jejaring Geopark Global Unesco (Unesco Global Geopark Network), Geopark Indonesia, dan Geopark Global Unesco Batur.

Ruang Kebumian

Ruangan ini berada di lantai I dimulai dengan panel Alam Semesta dan Pembentukan Planet Bumi, Terjadinya Tenaga Tektonik dan Gunung api Fenomenal di Indonesia.

Ruang Hayati

Ruang Hayati menyajikan koleksi tanaman yang dikeringkan (Herbarium) didalam gedung (indoor exhibition) dan tanaman hidup (living collection) diluar gedung (outdoor collection).

Ruang Kebudayaan

Penataan koleksi kebudayaan berdasarkan tema-tema Habitasi, Konsepsi Gunung dalam Kebudayaan, Interaksi Manusia dengan Alam, dan Diversitas Wujud Budaya Bali.

Museum Geopark Batur yang terletak di Penelokan, Kecamatan Kintamani, Kabupaten Bali merupakan pengembangan dari Museum Gunung api Batur. Pembangunan Museum Gunung api Batur terutama dilatarbelakangi oleh banyaknya gunung api di Indonesia sebagai negara yang di jalur cincin api (ring of fire) dunia, dengan tujuan menyampaikan informasi gunung api Indonesia, baik ditinjau dari aspek bahaya maupun manfaatnya, kepada masyarakat.

Proses pembangunan Museum Gunung api Batur membutuhkan waktu yang cukup lama melalui proses kerjasama tiga pihak, yaitu Kementerian Energi dan Sumber daya Mineral, Pemerintah Provinsi dan Kabupaten Bangli, serta Kementerian Kehutanan. Peletakan batu pertama pembangunan Museum Gunung api dilaksanakan pada 26 Maret 2004 oleh Direktur Jenderal Geologi Sumber Daya Mineral, Kementerian Energi dan Sumber Daya Mineral, Guber??nur Bali dan Bupati Bangli dan diresmikan oleh Menteri Energi dan Sumber Daya Mineral pada tanggal 10 Mei 2007. Pada awalnya Museum Gunung api Batur dikelola oleh Pemerintah Kabupaten Bangli, hingga kemudian pengelolaannya diserahkan kepada Badan Geologi, Kementeriaan ESDM pada tahun 2011.

Museum Geopark Batur mulai dibangun setelah kawasan Kaldera Batur diterima sebagai anggota jejaring geopark global (Global Geopark Net??work) bulan September 2012. Secara singkat, pembangunan Museum Geopark Batur dilakukan secara bertahap sebagai berikut:

  • 1. Tahap perencanaan: Juni ??? Agustus 2013.
  • 2. Tahap pembangunan fisik gedung: Agustus ??? Desember 2013.
  • 3. Tahap perencanaan interior dan sistem pameran : Maret 2014 ??? Mei 2014.
  • 4. Tahap pembangunan fisik interior : Juni 2014 ??? Desember 2014.
Dengan telah selesainya pembangunan Museum Geopark Batur, maka Museum Gunung api Batur melebur kedalam Museum Geopark Batur, dimana informasi aspek kebencanaan dan manfaat gunung api Batur bertindak sebagai peragaan utama keragaman geologi kegunungapian di dalam Museum Geopark Batur
